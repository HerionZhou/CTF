其他一些基本的表达：
 * false       =>  ![]
 * true        =>  !![]
 * undefined   =>  [][[]]
 * NaN         =>  +[![]]
 * 0           =>  +[]
 * 1           =>  +!+[]
 * 2           =>  !+[]+!+[]
 * 10          =>  [+!+[]]+[+[]]
 * Array       =>  []
 * Number      =>  +[]
 * String      =>  []+[]
 * Boolean     =>  ![]
 * Function    =>  []["filter"]
 * eval        =>  []["filter"]["constructor"]( CODE )()
 * window      =>  []["filter"]["constructor"]("return this")()
