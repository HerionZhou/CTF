
To run GPU recovery you need:

- NVidia graphic card with CUDA compute capability 2.0 or higher, CUDA-supported device drivers
- AMD Radeon cards with OpenCL v1.2 support (Radeon HD 7xxx or higher), Catalyst drivers 15.12 or newer
- You will need to install latest drivers in order the program to bu able to support Intel Graphics cards
