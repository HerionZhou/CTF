### **函数原型**

```c
#include<ctype.h>
int isprint(int c);
```



### **功能描述**

检查给定的字符能否被打印，即为数字、大写字母、小写字母、标点符号或空格，或任何当前C本地环境分类为可打印字符。





### **使用示例**

打印字符串中所有可打印字符。

```c
#include<stdio.h>
#include<stdlib.h>
#include<ctype.h>

int main(int argc,char *argv[])
{
	char *str = "test \n demo \0";
	int len = strlen(str);
	int i;
	for(i =0 ; i<len ;i++)
	{
		if(isprint(str[i]))
		{
			printf("%c",str[i]);
		}
	}
	printf("\n");
	return 0;
}
```

运行结果：

```c
test  demo 
```

通过运行结果可以看出\n和\0不是可打印字符，所以不会输出。





### **参数说明**

int c ------ 要检测的的字符，也可以是EOF。





### **函数返回值**

 - 若字符为可打印字符，则返回会非零
 - 若字符为不可打印字符，则返回零






### **使用注意事项**

- 哪些字符属于哪个类的详细信息取决于当前的语言环境
